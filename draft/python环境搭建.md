## python 安装
- 选择2.7版本 brew install python@2
- 建立虚拟环境 pip install virtualenv

## 建立工程
- 创建工程名 mkdir project_name
- 进入工程名后 建立python虚拟运行环境 virtualenv venv
- 进入虚拟环境 source venv/bin/activate
- 安装所需要的库 pip install -r requirements.txt
- 离开 deactivate
