# 版本历史

## v1.1.8 (正在开发特性)
- 看大图鼠标滚轮操作的优化
  

## v1.1.7 （2020.10.04)
- 新增素材收集浏览器插件
- 支持网页上的图片拖拽收集
- 支持网页整页图片批量收集
- 支持网页整页截图收集
- 支持右键菜单、快捷键收集
- 优化USB鼠标连接时的界面
- 优化自适应、瀑布流布局
- 新增对webp图片格式的支持
- 修复一些问题


## v1.1.6（2020.09.07)
- 增加批量重命名
- 增加导出到计算机
- 增加素材移动选择目标文件夹功能
- 增加管理文件夹拖拽排定次序
- 增加素材右键菜单
- 增加相册右键菜单
- 增加管理文件夹右键菜单
- 增加左侧导航条目图标及图标颜色设置功能
- 修复一些问题


## v1.1.5（2020.08.23)
- 增加导出picseepack素材包功能
- 增加导入picseepack素材包功能
- 增加导入eagle素材包功能
- 增加导入花瓣面板素材功能
- 优化侧边导航栏，可改变宽度，可搜索条目
- 优化素材选中的样式，更加美观
- 修复一些问题


## v1.1.4（2020.08.08)
- 重构图片整理功能
- 支持建立标签体系，为图片打标签，快速定位图片
- 支持文件夹整理，多层级分类归类
- 支持建立智能文件夹，按名称、标签、颜色等条件自动分类图片
- 支持按颜色、关键字、形状、尺寸、类型等条件查找筛选图片
- 支持自适应、瀑布流、网格布局浏览图片
- 支持智能任务相似图片的筛选
- 支持标签、注释、评分等信息的iCloud同步、多机协同
- 源码上模块化图片整理、屏幕截图、图片查看、图片编辑、图片导出
- 优化性能、修复一些问题
- 老版本升级重置用户数据


## v1.1.3（2020.01.21)
- 优化浏览图片、编辑图片界面
- 修复Retina屏幕下窗口截图、滚动截图问题

## v1.1.2（2020.01.01)
- 新增浏览图片时，画中画缩略导航
- 新增浏览图片时，显示图片元信息
- 版本升级会重置用户数据，包括标签库等，所以升级慎重
- 模块化, Framework优化工程Picsee
- 统计分析由Fabric切换到appcenter


## v1.1.1（2019.10.22)
